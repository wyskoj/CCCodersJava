The else & else if blocks

We can use the else if block to make another comparison if the first one is false.

    if (x == 2) {
        // Runs only if x is 2
    } else if (x == 3) { // x is not 2, maybe it's 3?
        // Runs only if x is 3
    } else {
        // Runs only if x is not 2 or not 3
    }
