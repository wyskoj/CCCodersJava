BMI Calculator

In this task, we'll create a BMI calculator (body mass index). It is calculated from the person's weight and height in kilograms and meters.

A person's BMI is calculated by \(\frac{W}{H^2}\). Once we calculate this value, we can output a string describing the person.

BMI Category
< 18.5 Underweight
18.5-24.9 Healthy
25 - 29.9 Overweight
30+ Obese
To finish this task, complete each /* TODO */.
  1. Getting the user input for weight and height
    2. In this task, the Scanner has been created for you. Use it to collect some input. Remember when we used Scanner to get text? We used the .nextLine() function. In this case, we want to collect decimal numbers (double). Therefore, we can use the .nextDouble() function.
  2. Calculating the BMI
    3. We calculate the BMI by dividing the weight by height2. To divide two numbers, we can use the / operator. To raise a number to another power, use the Math.pow() function. The function wants two numbers inside its parenthesis. The first number is the lower number. The second number is the exponent. Separate the numbers with a comma.

    Remember, even though weight and height are variables, they act like numbers, so you can write them inside the parenthesis.
  3. Printing the BMI category
    4. To print the BMI category, use a chain of if, else if, and else. Your first check should see if the BMI is less than 18.5 (look at the table above). If it is, you should print "Underweight". If the conditional is false, check to see if it less than 25. You can write that like this:
